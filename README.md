## TDD Shortcut Cheatsheet in IntellijI

1. 打开项目视图：`ALT` + `1`
2. 上下键切换到`src/test/java`目录
3. 新建Test Class：`ALT` + `Insert`
4. 移动到行尾：`End`
5. 自动修复错误：移动光标到错误处，按下`ALT` + `Enter`
6. 格式化代码：`CTRL` + `ALT` + `L`
7. 切换类/文件：`CTRL` + `E`，上下键选择文件
8. 运行单元测试：`CTRL` + `SHIFT` + `F10`
9. 切换命令行窗口：`ALT` + `F12`

## Example2: FizzBuzz

报数游戏：

1. 按自然数报数
2. 如果该数可以被3整除，报“Fizz”
3. 如果该数可以被5整除，报“Buzz”
4. 如果既可以被3整除，又可以被5整除，报“FizzBuzz”
